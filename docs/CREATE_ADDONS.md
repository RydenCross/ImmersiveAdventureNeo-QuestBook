# Create Addons

The Create Addons chapter expands the completed core Create progression into the addon ecosystem installed in Immersive Adventure Neo.

Covered integrations include:

- Create: New Age
- Create: Diesel Generators
- Create: Big Cannons
- Create: Connected
- Create Better Motors
- Create Mechanical Companion
- Create Contraption Terminals
- Sophisticated Storage in Motion
- Tracks
- Bells & Whistles
- Compact Gearbox
- Linear Bearing
- Create Aeronautics
- Create Propulsion
- Create Thrusters
- Aeroworks

Because several of these projects change registry names between builds, version-sensitive objectives use checkmark tasks and describe the functional build goal instead of relying on fragile item IDs.
